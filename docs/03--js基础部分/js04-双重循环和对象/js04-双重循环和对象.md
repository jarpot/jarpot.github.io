### 1.3 break关键字 

break:表示中断程序的执行。中断当前所在的for循环。

```
//相当于给外层循环命名为aaa
aaa:for(var i=1;i<=9;i++){
				for(var j=1;j<=i;j++){	
					if(i==5){
						break aaa;//单个break表示中断当前所在的for循环  
						//表示 中断名字为aaa的for循环
					}
					document.write(j+"x"+i+"="+i*j+"&nbsp;&nbsp;");	
				}
				document.write("<br/>");
			}
```

### 1.4 continue关键字 

继续：中断当次循环，继续进行下一次的循环。

```
//continue 继续
			for(var i=1;i<=9;i++){
				for(var j=1;j<=i;j++){	
				
					if(j==5){
						continue;
					}
					document.write(j+"x"+i+"="+i*j+"&nbsp;&nbsp;");
					
				}
				document.write("<br/>");
			}
```
